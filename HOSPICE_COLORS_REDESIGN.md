The **Purple Butterfly** is used globally in healthcare to indicate end-of-life care:

```css

--hospice-lilac: #C8A2D0
--hospice-lavender: #E6D5F0

--sunlight-yellow: #FFD966

```

### **Animations**
- `hospiceFlow` - 25s background gradient flow
- `butterflyPulse` - 4s logo pulse with rotation
- `floatButterfly` - 20s floating pattern movement
- `butterflyFloat` - 3s gentle floating effect

- **Main Background:** Purple → Lilac → Lavender
- **Login:** Purple → Lilac → Green
- **Buttons:** Purple → Green on hover
- **Cards:** Cream → Light Lavender
- **Metrics:** Orange → Yellow, Purple → Lilac, Green → Soft Green
